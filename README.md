# Fundamentos Da Web

Projeto de exercícios de aprendizado de HTML/CSS em português.

## Estrutura do projeto

- `index.html` - página principal
- `produtos.html` - página de produtos
- `contato.html` - página de contato
- `style.css` - estilos compartilhados
- `imagens/` - recursos de imagem
- `videos/` - recursos de vídeo

## Como usar

1. Abra `index.html` em um navegador.
2. Navegue pelas páginas usando os links do site.

## Desenvolvimento

- Edite os arquivos HTML/CSS conforme necessário.
- Recarregue a página no navegador para ver as mudanças.
